qq 音乐

### 在线预览

> 可能加载有点慢，服务器在国外🤣====> [手机预览](https://shenzekun.github.io/QMusic)

### 安装方法

打开终端 ，输入：

```bash
git clone git@github.com:shenzekun/QMusic.git
```

切换到克隆下来的文件夹：

```bash
cd QMusic
```

然后输入：

```bash
npm install
```

在浏览器里打开`index.html`
